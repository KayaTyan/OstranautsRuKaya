## 🎮 О локализации

**OstranautsRuKaya** — неофициальная фанатская локализация игры [Ostranauts](https://bluebottlegames.com/games/ostranauts) (Blue Bottle Games) на русский язык. Проект не связан с разработчиком и является самостоятельной работой сообщества. Включает полный перевод игровых данных и BepInEx-плагин, который обеспечивает корректное отображение русской грамматики, спряжения глаголов и динамических текстовых конструкций.

В основе проекта лежат наработки двух локализационных сообществ, объединённые и переработанные в единый, более надёжный продукт.

## ✨ Возможности

### Грамматический движок

- **Полное спряжение глаголов по лицам** — более 2 300 глаголов с формами для каждого лица и числа (я говорю / ты говоришь / он говорит / мы говорим / вы говорите / они говорят)
- **Русские местоимения** — корректные субъектные, объектные, притяжательные и возвратные формы для всех типов существительных
- **Конструкции «у него / у неё / у них»** — правильный перевод английского has для русского языка
- **Автоматическое удаление английских артиклей** — «the door» → «дверь», без «the» в начале
- **Fallback для пропущенных токенов** — перехват `DataHandler.PrepareToken` гарантирует, что ни один глагол не останется необработанным

### Перевод контента

- **400+ переведённых JSON-файлов** игровых данных — строки интерфейса, описания предметов, взаимодействия, диалоги, сюжеты, условия, квесты
- **Все сюжетные линии** — включая полностью переведённый сценарий Ceres (235 текстов, 100%)
- **Обучение (TutorialBeat)** — все обучающие цели переведены во внешнем JSON-файле с поддержкой подстановки клавиш
- **Логика Condition Owners** — перевод описаний и названий всех владельцев условий, предметов и модулей
- **Сюжетные триггеры** — корректная обработка `plot_beat_overrides` и `interaction_overrides` с сохранением match-ключей

### Технические патчи

- **RestoreNavStation** — исправлена проверка завершения обучения: используется стабильное поле `strDuty` вместо локализованного заголовка
- **ObjectivePanel.RefreshText** — принудительное обновление текста целей гарантирует отображение перевода
- **CondOwner.LogMessage** — фильтрация английских грамматических конструкций в системных сообщениях

### Архитектура

- **Минимум Harmony-патчей** — всего 8 точек перехвата вместо 200+ при хардкоде, что снижает риск конфликтов с обновлениями игры
- **Внешние таблицы** — переводы глаголов и обучающих целей вынесены в JSON-файлы, что позволяет обновлять переводы без перекомпиляции
- **Безопасные патчи** — все перехваты обёрнуты в try-catch, что исключает краши игры при ошибках

## ⚙️ Как работает плагин

### Спряжение глаголов

Оригинальный движок `GrammarUtils` различает только базовую и 3-ю форму глагола. Для русского языка этого недостаточно — необходимо спряжение по шести формам.

Плагин перехватывает `GrammarUtils.Verb` и выбирает нужную форму на основе `SentenceEntity.InflectionIndex` субъекта:

```
InflectionIndex 0 (1-е лицо)  → formIdx 0 (я делаю)
InflectionIndex 1 (2-е лицо)  → formIdx 1 (ты делаешь)
InflectionIndex 2-3 (3-е л.)  → formIdx 2 (он/она делает)
InflectionIndex 4 (мн.ч.)     → formIdx 5 (они делают)
```

Данные спряжения загружаются из `verb_conjugations.json`:

```json
{
  "infinitive": "говорить",
  "forms": ["говорю", "говоришь", "говорит", "говорим", "говорите", "говорят"]
}
```

### Удаление артиклей

Оригинальный движок добавляет артикль `the ` перед названиями неодушевлённых предметов (`the door`, `the reactor`). Плагин перехватывает `GrammarUtils.AttemptProperName` и удаляет английский артикль, оставляя только название предмета.

### Обучающие цели

Тексты встроенных обучающих целей (`TutorialBeat`) вынесены во внешний файл `tutorial_translations.json`. Плагин перехватывает создание и обновление целей, подставляя переведённые тексты. Маркеры `{glyph:...}` автоматически заменяются на актуальные клавиши управления через `InputManager`.

### RestoreNavStation

Игра проверяет `strTitle == "Restore"` для завершения обучающей цели «Восстановить навигационную станцию». После перевода названия в «Ремонт» эта проверка ломается. Плагин использует стабильное поле `strDuty == "Restore"` как запасной вариант.

## 🔧 Сборка из исходников

### Требования

- Mono / .NET SDK
- DLL-библиотеки из игры (`Assembly-CSharp.dll`, `BepInEx.dll`, `0Harmony.dll`, `UnityEngine.dll`)

### Компиляция

```bash
mcs -target:library \
    -out:OstranautsRuKaya.dll \
    -r:lib/0Harmony.dll \
    -r:lib/Assembly-CSharp.dll \
    -r:lib/BepInEx.dll \
    -r:lib/netstandard.dll \
    -r:lib/UnityEngine.dll \
    -r:lib/UnityEngine.CoreModule.dll \
    -r:lib/UnityEngine.UI.dll \
    -r:lib/MonoMod.RuntimeDetour.dll \
    -r:lib/MonoMod.Utils.dll \
    src/Main.cs
```
